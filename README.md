# Typing Website

A typing speed test web application built with HTML, CSS, and JavaScript.

## Features
- Measures typing speed (WPM - Words Per Minute)
- Tracks accuracy
- Simple and clean UI

## How to Use
1. Open the website
2. Start typing the displayed text
3. Your WPM and accuracy will be calculated automatically

## Tech Stack
- HTML
- CSS
- JavaScript
